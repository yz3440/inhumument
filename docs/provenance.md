# Provenance & License

## The book

_A Human Document_ by **W. H. Mallock** — the one-volume **Chapman & Hall 1892
"New Edition"**, the edition **Tom Phillips** treated to make _A Humument_. Page
numbers map one-to-one onto _A Humument_, so a page here is the **printed book
page (= the _A Humument_ page), 1–367**.

The source scan is [Internet Archive item
`ahumandocumenta04mallgoog`](https://archive.org/details/ahumandocumenta04mallgoog),
digitized by Google from a public-domain copy. The OCR (Apple Vision) and the
CV pipeline that produced the dataset live in the
[**humument**](https://github.com/yz3440/humument) repo; the data is published
as the `humument-data` and `humument-images` npm packages.

## The packages

- [`humument`](https://www.npmjs.com/package/humument) — renderer-agnostic core.
- [`humument-data`](https://www.npmjs.com/package/humument-data) /
  [`humument-images`](https://www.npmjs.com/package/humument-images) — the dataset.
- [`inhumument-lib`](https://www.npmjs.com/package/inhumument-lib) — the p5
  rendering layer (this project).
